# nju-alley-server

微信小程序开发大赛作品后端。

## 项目简介

nju-alley，具有直观地理联系的校园实时互动社区。

它将助你创造只属于你的nju；它将带你认识南大人心中的nju；它将向所有人呈现真实的nju。
